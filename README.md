# 📄 MarinaDoc — система автоматизации документов

MarinaDoc — это desktop-приложение для автоматического формирования документов на основе данных из исходного договора.

Программа извлекает ключевые данные, структурирует их и формирует готовый комплект документов в форматах Word и Excel.

---

## 🚀 Польза для бизнеса

- ускорение обработки документов до 90%
- снижение количества ошибок при ручном вводе
- автоматизация рутинных задач
- экономия времени сотрудников

## 📌 Что делает программа

Типовой сценарий работы:

1. загрузка исходного документа
2. извлечение ключевых данных
3. проверка и редактирование в интерфейсе
4. генерация документов
5. сохранение результата

## 🏗️ Структура проекта

```text
MarinaDoc/
├── app/                 # исходный код приложения
│   ├── core/            # конфигурация, контроллер, логирование
│   ├── models/          # модели данных
│   ├── services/        # бизнес-логика
│   └── ui/              # интерфейс (PySide6)
├── assets/              # скриншоты и медиа
├── config/              # конфигурация
├── docs/                # документация
├── examples/            # demo-примеры
├── resources/           # ресурсы приложения
├── templates/           # шаблоны документов
├── tests/               # тесты
├── main.py              # точка входа
└── requirements.txt     # зависимости
```
